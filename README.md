# Customer_Behavior_Analysis

## Overview
This is an end-to-end data analytics project focused on analyzing customer shopping behavior using Python, SQL, PostgreSQL, and Power BI.

The project covers the complete analytics workflow, including data cleaning, exploratory data analysis (EDA), SQL-based business analysis, dashboard building, and reporting.

The main goal of this project is to identify customer purchase patterns, spending trends, product performance, and customer preferences.

---

## Dataset
The dataset contains customer shopping behavior information such as:

- Customer ID
- Age
- Gender
- Item Purchased
- Category
- Purchase Amount
- Location
- Review Rating
- Shipping Type
- Payment Method
- Discount Applied
- Frequency of Purchase

Dataset was used for analysis and dashboard creation.

## Project Workflow

### 1. Data Loading
- Imported CSV dataset into Python using Pandas.

### 2. Data Cleaning
- Checked missing values
- Renamed columns
- Removed unnecessary columns
- Standardized column names

### 3. Exploratory Data Analysis (EDA)
Performed analysis on:
- Customer demographics
- Purchase trends
- Product categories
- Revenue patterns
- Rating analysis

### 4. SQL Analysis
Loaded cleaned data into PostgreSQL and performed SQL queries such as:
- Top-selling products
- Average review ratings
- Revenue by category
- Customer purchase behavior by location
- Shipping method analysis

### 5. Dashboard Creation
Built an interactive Power BI dashboard for visual storytelling and business insights.

### 6. Reporting
Created project report and presentation using Gamma.
